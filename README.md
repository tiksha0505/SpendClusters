# SpendClusters
## It is a combination of MERN and ML concepts

Most expense trackers just tell you where your money went (e.g., 'Food' or 'Travel'). But after finishing my 4th semester, I wanted to build something that tells me how I spend. Categories are labels, but clusters are patterns. I built SpendClusters to find the hidden 'spending personas' in my own data

I chose the MERN stack for its scalability. 
The frontend (React) handles the visualization via Recharts, while the backend (Node/Express) serves as the bridge to the clustering engine. 

The ML logic runs on the server side to ensure that as the user adds more transactions, the clusters evolve in real-time.
