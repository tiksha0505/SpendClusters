# SpendClusters 💰📍
### *Where Math Meets Money: A Behavioral Expense Tracker*

**SpendClusters** is a specialized MERN-stack application designed to uncover "spending personas" hidden in financial data using **K-Means Clustering**.

---


Most expense trackers just tell you **where** your money went (e.g., 'Food' or 'Travel'). After finishing my 4th semester, I wanted to build something that tells me **how** I spend. 

Categories are labels, but **clusters are patterns**. I built this to bridge the gap between theoretical Machine Learning concepts and practical, scalable Web Development.

---

## Techs Used: 
*   **Frontend:** React.js (Component-based UI) & Recharts (Data Visualization)
*   **Backend:** Node.js & Express.js (RESTful API Architecture)
*   **Database:** MongoDB (Scalable Document Storage)
*   **ML Engine:** Centroid-based K-Means Clustering

---

## WorkFlow: 
1.  **Data Collection:** Users log transactions via a clean, responsive MERN interface.
2.  **Behavioral Analysis:** The server-side logic analyzes the `amount` and `frequency` of entries.
3.  **Clustering:** The K-Means algorithm mathematically groups data into "zones" (e.g., *Micro-Habits* vs. *Heavy-Hitters*).
4.  **Visual Feedback:** A color-coded scatter plot in React reflects spending clusters in real-time.
